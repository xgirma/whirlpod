# Whirlpod

Routes

## GET /api/
Return "OK"

## GET /api/pods
Return all pods.

## GET /api/pods/ten/:type
Return ten recent or liked podcasts.

## GET /api/pods/:title/:type
Return ten recent or liked pods of a playlist.

## POST /api/pods/:id
Increment like by one.

## POST /api/pods
Post multiple pods.
